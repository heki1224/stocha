# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.2.x   | ✅        |
| < 1.2   | ❌        |

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub Issues.

Instead, open a [GitHub Security Advisory](../../security/advisories/new) to report
the issue privately. We will review it on a best-effort basis and aim to address
confirmed vulnerabilities as quickly as possible.

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce (proof-of-concept code if applicable)
- Affected version(s)

We appreciate responsible disclosure.
